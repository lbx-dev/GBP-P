A library for ethereum smart contract development

## Tech stack

-   truffle framework
-   OpenZepplin 2.0

## Goals

-   ERC-20 token
-   [Ownable (from OpenZepplin)][oz-ownable]
-   [ERC20Detailed (from OpenZepplin)][oz-erc20detailed]
-   Transferrable
-   [ERC-621][erc-621]
-   [ERC-223][erc-223] (aspirational)
